Apa itu Desain Web Responsif?

Desain Web Responsif adalah tentang penggunaan HTML dan CSS untuk secara otomatis mengubah ukuran, menyembunyikan, mengecilkan, atau memperbesar situs web, agar terlihat bagus di semua perangkat (desktop, tablet, dan ponsel):

Mengatur Viewport

Untuk membuat situs web responsif, tambahkan tag berikut <meta> ke semua halaman web Anda:

Contoh

<meta name="viewport" content="width=device-width, initial-scale=1.0>

Ini akan mengatur viewport halaman Anda, yang akan memberikan instruksi kepada browser tentang cara mengendalikan dimensi dan skala halaman.

Berikut adalah contoh halaman web tanpa tag meta viewport, dan halaman web yang sama dengan tag meta viewport:

Gambar Responsif

Gambar responsif adalah gambar yang skalanya dapat disesuaikan dengan ukuran browser apa pun.

Menggunakan Properti Lebar

Jika properti CSS width ditetapkan ke 100%, gambar akan responsif dan diskalakan ke atas dan ke bawah:

Contoh

Code

<img src="img_girl.jpg" style="width:100%;">

Perhatikan bahwa dalam contoh di atas, gambar dapat diperbesar hingga lebih besar dari ukuran aslinya. Solusi yang lebih baik, dalam banyak kasus, adalah menggunakan max-widthproperti tersebut.

Menggunakan Properti max-width

Jika max-widthp roperti ditetapkan pada 100%, gambar akan diperkecil jika diperlukan, tetapi tidak akan pernah diperbesar hingga lebih besar dari ukuran aslinya:

Code

<img src="img_girl.jpg" style="max-width:100%;">

Ukuran Teks Responsif

Ukuran teks dapat diatur dengan satuan "vw", yang berarti "lebar tampilan".Dengan cara itu ukuran teks akan mengikuti ukuran jendela browser:

Hello World!

contoh

<h1 style="font-size:10vw;">Responsive Text</h1>
<p style="font-size:5vw;">Resize the browser window to see how the text size scales.</p>
<p style="font-size:5vw;">Use the "vw" unit when sizing the text. 10vw will set the size to 10% of the viewport width.</p>
<p>Viewport is the browser window size. 1vw = 1% of viewport width. If the viewport is 50cm wide, 1vw is 0.5cm.</p>

hasil

Responsive Text

Resize the browser window to see how the text size scales.

Use the "vw" unit when sizing the text. 10vw will set the size to 10% of the viewport width.

Viewport is the browser window size. 1vw = 1% of viewport width. If the viewport is 50cm wide, 1vw is 0.5cm.
