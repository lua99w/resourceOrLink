HTML Layout Elements

nah jadi di layout html ini juga ada element element penting nya dan itu ada gunanya masing masing

  • <header> digunakn untuk menentukan bagian header atau bagaian awal sebuah web
  • <nav> digunakn untuk menentukan bagian navigati atau link reques sebuah web
  • <section> Untuk membagi halaman menjadi bagian tematik, sering disertai dengan heading (<h1>, <h2>, dll.)
  • <article> Untuk konten yang dapat berdiri sendiri, seperti artikel atau postingan yang dapat dipublikasikan
  • <aside> Untuk konten tambahan yang relevan tapi tidak terlalu penting atau utama
